## NFTチャンネルの作り方

### 材料
- 浅型万能トレー or フタ付き収納BOX（チャンネル本体）
- 黒プラダン（フタ）
- プラカップ（ネットポット代替）
- スポンジ（培地）
- グルーガン（防水加工）

### 手順

#### 1. フタにネットポット穴を開ける

黒プラダンを容器のサイズに合わせてカットし、等間隔で穴を開ける。

```
┌────────────────────────────────────┐
│  ○    ○    ○    ○    ○    ○   │
│ 3-5cm径  間隔5cm                   │  ← 黒プラダン
│                                    │
│  ●給水穴(6mm)         ●排水穴(10mm)│
└────────────────────────────────────┘
```

- 穴のサイズ: プラカップの口径に合わせる（3-5cm）
- カッターで円形に切り抜く（コンパスカッターがあると便利）
- 未使用の穴は黒テープで塞ぐ

#### 2. 給水口・排水口を作る

```
給水側（高い方）:
  フタの端に6mmの穴 → チューブ差し込み

排水側（低い方）:
  容器の底近くに10mmの穴 → チューブ接続
  穴の周りをグルーガンで防水シール
```

#### 3. 傾斜をつける

```
窓側(高い)                    室内側(低い)
┌──────┐
│割り箸│
│ 2本  │
├──────┼──────────────────────────────┐
│      │  チャンネル容器              │
└──────┴──────────────────────────────┘
↑ +5mm                               ↑ 0mm
棚板 ═════════════════════════════════
```

25cmの奥行きに対して5mmの高低差 = 約2%の傾斜。
水を流して排水口に向かうことを確認。

#### 4. ネットポット（プラカップ）の加工

1. プラカップの側面に8-12箇所、底に4-6箇所の穴を開ける（径3-5mm）
2. はんだごて or 熱した釘で溶かして開けるときれい
3. カッターでも可（バリに注意）
4. スポンジを2-3cm角に切り、中央に十字の切り込みを入れて挿入
5. 上部にハイドロボールを入れて安定させる

**代替案**: ダイソーの「マヨケチャホルダー」（2個入り110円）は最初から穴が開いた構造で、ネットポットに非常に近い。

## 遮光加工の手順

藻は「光 + 栄養素 + 水」で繁殖する。完全な遮光が必須。

### リザーバー

不透明BOXなら最低限の加工でOK:
1. フタの隙間にアルミテープを貼る
2. チューブ・センサーの穴周りもテープで塞ぐ
3. 透明BOXの場合はアルミ保温シートを全面に巻く

### チャンネル

1. 黒プラダンのフタで上面を遮光
2. 容器が半透明なら外面をアルミホイルで包む
3. ネットポット穴の隙間にアルミテープ
4. 未使用の穴は黒テープで完全に塞ぐ

### チューブ

ダイソーのエアーチューブは透明なので:
1. 黒ビニールテープを巻く、または
2. アルミテープを巻く

## ダイソー エアーポンプ（酸素供給用）

ダイソーには770円のエアーポンプがある（乾電池式）:
- 単1乾電池1本で最大約40時間駆動
- 強弱2モード（強: 1.0L/分、弱: 0.7L/分）
- ホース100cm + エアストーン付属

ただし乾電池式のため常時稼働には不向き。
常時稼働させるならAmazonでAC電源式のエアーポンプ（1,000-2,000円）を推奨。
