## 組立手順（ステップバイステップ）

### STEP 1: メタルラックの組立

1. ポール4本を立てる
2. 棚板を以下の高さで固定:

```
90cm ── 上段棚板（チャンネルA）
         間隔: 約25cm
65cm ── 中段棚板（チャンネルB）
         間隔: 約30cm
35cm ── 下段棚板（リザーバー+機器）
         間隔: 35cm（床からの高さ）
 0cm ── 床（キャスター装着時+5cm）
```

3. 窓際に設置（ラック背面が窓側）
4. 水平を確認（百均のミニ水平器で可）

### STEP 2: 各段に防水トレイを敷く

1. 各棚板の上にダイソー浅型万能トレーを設置
2. ラック全体の下にレジャーシートを敷く（床保護）

### STEP 3: NFTチャンネルの製作

#### フタの穴開け

```
┌────────────────────────────────────┐
│  ○    ○    ○    ○    ○    ○   │
│ 3-5cm径  間隔5cm                   │  ← 黒プラダン
│                                    │
│  ●給水穴(6mm)         ●排水穴(10mm)│
└────────────────────────────────────┘
```

- 黒プラダンを容器サイズにカット
- カッターでネットポット穴を開ける
- 給水穴(6mm)と排水穴(10mm)も開ける

#### 排水口の防水処理

```
容器の排水口:
┌──────────┐
│  壁面    │
│   ○ ←──── 10mmの穴にチューブを通す
│  壁面    │     穴の周りをグルーガンで防水
└──────────┘
```

#### 傾斜の設定

- 窓側の端に割り箸を2本挟んで+5mmの傾斜
- 水を少し流して排水口に向かうことを確認

### STEP 4: リザーバーの設置

1. 下段棚板の窓側に不透明BOXを設置
2. フタに穴を開ける:

```
┌────────────────────────────┐
│  ○ 給水チューブ通し穴(6mm)│
│  ○ 排水チューブ戻り穴(10mm)│
│  ○ ECセンサー通し穴       │  ← フタ
│  ○ 水温センサー通し穴     │
│  ○ ポンプ電源ケーブル穴   │
└────────────────────────────┘
```

3. 穴の周りをアルミテープで遮光

### STEP 5: ポンプと配管の接続

1. ポンプをリザーバーの底に設置
2. ポンプ吐出口にメインチューブ(6mm)を接続
3. T字分岐で2本に分岐（各チャンネルへ）
4. 各分岐にバルブ（流量調整用）を取付
5. 排水チューブをリザーバーのフタの穴へ接続

```
チューブの固定は柱に結束バンドで:

右の柱 = 水系（チューブ）
左の柱 = 電気系（センサー線・電源線）

※ 水と電気を物理的に分離すること！
```

### STEP 6: センサーとRaspberry Piの設置

1. DS18B20（水温）→ フタの穴からリザーバーの水中へ
2. ECセンサー → 同様にフタから水中へ
3. フロートスイッチ → リザーバー内壁にキスゴムで固定
4. Pi 4 → 下段棚板の室内側にジップロックに入れて設置
5. GPIO配線

```
Pi 4 GPIO配線:
  GPIO 4  → DS18B20 DATA（要4.7kΩプルアップ抵抗）
  GPIO 17 → リレーCH1 → ポンプ制御
  GPIO 27 → リレーCH2 → LED制御（増設時）
  GPIO 22 → フロートスイッチ（水位）
  I2C SDA → ADS1115 SDA → ECセンサー
  I2C SCL → ADS1115 SCL
```

### STEP 7: 遮光処理

全ての光が養液に当たらないようにする:

| 箇所 | 方法 |
|------|------|
| リザーバー | 不透明BOX + フタの隙間にアルミテープ |
| チャンネル | 黒プラダンのフタ + 外面にアルミホイル |
| チューブ | 黒ビニールテープを巻く |
| ネットポット周囲 | アルミテープで隙間を塞ぐ |
| 未使用穴 | 黒テープで完全に塞ぐ |

### STEP 8: 水漏れ確認チェックリスト

```
□ 1. リザーバーに水だけ入れて漏れ確認（養液はまだ）
□ 2. ポンプON → 水が出るか確認
□ 3. 両チャンネルに水が分岐して流れるか
□ 4. チャンネル内で水が傾斜に沿って排水口へ流れるか
□ 5. 排水チューブからリザーバーに戻るか
□ 6. 30分間連続運転テスト（全接続部を確認）
□ 7. リザーバー水位が安定するか
□ 8. Pi 4起動、センサー値が取得できるか
□ 9. 遮光漏れがないか（暗所でライト照射して確認）
□ 10. 水と電気の接触リスクがないか
□ 11. ラックが安定しているか（転倒防止）
```

## 安全に関する注意事項

### 漏電対策
- 電源タップは漏電ブレーカー付きを使用
- 水系と電気系の配線は必ず分離
- Pi 4やリレーは水がかからない位置に

### 水漏れ対策
- 窓際のため、窓枠の結露にも注意
- ラック下に防水トレイ + レジャーシート
- 初回稼働時は数時間監視

### 寝室特有の注意
- 換気に注意（湿度上昇→カビ防止）
- ポンプの動作音に注意（静音タイプ推奨）
- 夏場: 直射日光で養液温度上昇 → 遮光カーテンで調整
- 冬場: 窓からの冷気で養液温度低下 → 水槽用ヒーター検討
- 理想的な養液温度: 18-25度C

### メンテナンス頻度
- 養液交換: 週1回
- EC値・pH確認: 毎日（Pi 4で自動化）
- 根の目視確認: 週1回
- チャンネル内の藻清掃: 月1回
- チューブ接合部の点検: 月1回
